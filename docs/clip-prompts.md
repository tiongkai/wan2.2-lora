# Clip Prompt Log

Persistent record of the exact prompt behind each generated clip. **For dataset batches**
(fighting / self-injury / the 100-clip run) the canonical per-clip prompts live in the
`*_manifest.csv` next to the clips. This file logs the **one-off / iterated test clips** —
especially the carpark explosion experiments, whose shared manifest
(`fighting_i2v_base/fighting_i2v_manifest.csv`) gets overwritten each single-clip run.

Convention: all base Wan2.2 I2V, 832×480. `auto` = prompt generated by Qwen via `--auto-prompt`
(model noted); `manual` = hand-written `--prompt`.

### Lessons (auto-prompt)
- **Qwen2.5-VL** free-associates: "explode" → it added flames AND unwanted "wheels spinning."
- **Qwen3.5** is more literal/faithful (esp. with the no-invented-motion rule): it gives exactly
  what's in the idea — so put constraints AND effects in the idea ("**parked**… **fireball with
  flames and black smoke**"). It dropped fire when the idea only said "explodes in place."
